# Contribution Protocol

开源项目标准贡献流程指南 | A Practical Guide to Standard Open-Source Contribution Workflows

## 简介

公共项目不是个人笔记，每一次贡献都值得被认真对待。

本项目提供一套**通用的、可实践的**开源项目贡献流程标准，适用于所有开源项目，不局限于特定社区或技术栈。

## 内容涵盖

- 认识开源项目结构
- Fork 工作流（Fork → Clone → Branch → Commit → Push → PR）
- Commit 规范（Conventional Commits）
- Pull Request 标准
- 仓库所有权转移
- Fork 后的站点预览与部署
- SEO 与搜索引擎配置
- 常见错误与反面模式

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build
```
